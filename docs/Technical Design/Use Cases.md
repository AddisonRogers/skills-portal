### Strategic Lead (Matt Jennings)
- As a strategic lead, I want to see what skills are trending and in-demand, so I can align learning with market needs.
- As a strategic lead, I want to view who has which skills and at what level, so I can staff projects and proposals effectively.
- As a strategic lead, I want to define capability frameworks (not just roles), so people can explore internal mobility.
- As a strategic lead, I want to structure learning paths by role and goal (e.g., “Team Leader” or “Cloud Engineer”), so employees can grow strategically.
- As a strategic lead, I want to track progress toward certifications, so I know who is ready for client-facing or senior work.
- As a strategic lead, I want to offer rewards for learning progression, so I can reinforce desired behaviours.
- As a strategic lead, I want to surface internal success (Culture Amp, Bamboo, LinkedIn), so learning translates to recognition.

### Apprentice
- As an apprentice, I want to see a clear starting point for learning frontend/backend/AI, so I know where to begin.
- As an apprentice, I want to progress through a defined learning path with levels (Foundation, Intermediate, Advanced), so I feel I’m improving.
- As an apprentice, I want to unlock rewards, badges, or certificates, so I stay motivated.
- As an apprentice, I want to toggle interest in certain skills or technologies, so I can be matched to client work I’m excited about.
- As an apprentice, I want to set goals with my team lead and check in regularly, so I get guidance and accountability.
- As an apprentice, I want to see how many hours a module will take and what I need to know, so I can plan effectively.
- As an apprentice, I want to access soft skills content (e.g., requirements gathering), so I’m ready for all aspects of a role.
- As an apprentice, I want to get suggestions based on what I’m currently learning, so I don’t feel overwhelmed by too many choices. 

### Graduate
- As a graduate, I want to know what skills I need to progress from Associate to Senior, so I can target my learning effectively.
- As a graduate, I want to see how my learning impacts my career and pay, so I understand its value.
- As a graduate, I want to view learning goals set by my team lead, so I stay aligned with expectations.
- As a graduate, I want to explore new capabilities, so I can consider shifting roles internally.
- As a graduate, I want to get recommended resources from my buddy or lead, so I don’t have to search from scratch.

### Placement Student
- As a placement student, I want to complete small learning challenges daily, so I can stay engaged without burnout.
- As a placement student, I want to track my streaks and milestones, so I feel accomplished.
- As a placement student, I want to earn points for getting questions right, so I can feel competitive and rewarded.
- As a placement student, I want to track progress visually, so I stay motivated.
- As a placement student, I want to avoid bloated or overly broad qualifications, so I can learn what’s truly useful.

### Regular Employees
- As an employee, I want to track what I’m currently learning and what I’ve completed, so I stay organised.
- As an employee, I want to choose from suggested learning paths for my current or future role, so I don’t waste time.
- As an employee, I want to explore trending skills, so I stay competitive in the industry.
- As an employee, I want to bookmark resources I hear talked about, so I can come back to them later.
- As an employee, I want to log self-directed learning (Pluralsight, MS Learn, YouTube, etc.), so informal effort is still tracked.

### System & Gamification Features
- As a user, I want to see recommendations based on my current learning path, so I stay focused.
- As a user, I want to receive nudges if I haven’t engaged in a while, so I don’t lose momentum.
- As a user, I want to see what other users are learning or interested in, so I feel part of a community.
- As a user, I want to see AI-generated insights, like “50% of people are learning X”, so I feel guided by data.
- As a user, I want to track my skill levels and visualise growth, so I can see how far I’ve come.
- As a user, I want to select a target career path, so my learning is aligned to my aspirations.
- As a user, I want to access sandbox environments, so I can practise safely.
- As a user, I want to print a certificate or share success on LinkedIn, so I can showcase progress externally.
